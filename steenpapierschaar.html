STEEN PAPIER SCHAAR
GIT

Het spel steen papier schaar vond ik wel een leuke toevoeging aan de site. Het is leuk om te maken en het is ook een goede manier om javascript te leren vind ik. Het idee was ook om het spel er mooi uit te laten zien door middel van het design van de site te gebruiken.

Eerst heb ik nagedacht over hoe ik het spel het meest efficiënt kan maken. Naar mijn mening kan ik hievoor het beste switch gebruiken. Ook heb ik een aantal variabelen toegevoegd over de keuze van de gebruiker en een random generator die kiest tussen 0, 1 of 2. Nadat het spel werkte heb ik plaatjes en een scorebord toegevoegd.

REFLECTIE

Ik heb geleerd hoe je en switch gebruikt in javascript. Ook heb ik geleerd hoe je de style van bepaalde id's in css kan aanpassen met javascript.
