NAVIGATIEBAR PORTFOLIO
Voor development wilde ik een betere navigatie maken voor de site. Zo kun je makkelijker bij de inviduele POCs en ziet het er mooier uit. Ik heb online gekeken hoe dat in zâ€™n werk gaat. Dit project in codepen heb ik gebruikt om te kijken hoe het werkt:

https://codepen.io/RajRajeshDn/pen/dyGZPMR

Na het kijken in de code is het na een paar uur gelukt om het te laten werken. Ik wilde niet precies de code copy pasten, maar er een eigen draai aan geven. Het belangrijkste is dat als de gebruiker klikt op aMain, dan veranderd de naam van de class naar aMain active. Hierdoor wordt de display in de style veranderd van none naar block. Hierdoor wordt de submenu zichtbaar. Ik weet nu hoe dat het werkt en nu zou ik het later nog een keer kunnen gebruiken.
Hierna ben ik begonnen met het verbeteren van de navigatiebalk. Het was hiervoor niet mogelijk om een submenu te sluiten door er buiten te klikken. Door wat hulp van Pim is dat uiteindelijk gelukt met behulp van deze bron:

https://techstacker.com/close-modal-click-outside-vanilla-javascript/#javascript-detect-all-clicks-on-document

Door deze extra functie is de site gebruikersvriendelijker.
