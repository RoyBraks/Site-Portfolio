JSON



GIT

Voor development heb ik gekeken naar hoe je een JSON file uitleest in javascript.

Door deze techniek kun je makkelijk data inladen en laten zien op je website. Het is redelijk eenvoudig te doen. Nadat ik de knop aan de praat kreeg, heb ik gekeken naar hoe ik het qua UX kan verbeteren. Eerst kon je oneindig op de knop klikken en kreeg je de data in één rits achter elkaar. Daarom heb ik alles in een list gezet in een div met display flex. Ook heb ik in de code gezet dat als je op de knop klikt eerst alle html in de div "jsonwrite" wordt verwijderd zodat je telkens maar één rij van de data ziet. Dit is mooier en beter voor de UX.

REFLECTIE

Met deze POC heb ik geleerd wat een JSON file is en hoe dat deze kan worden gebruikt in een site. Ook weet ik nu dat het redelijk gemakkelijk gaat.
